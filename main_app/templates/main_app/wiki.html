About

Maze Creator (Github repository) is a website to create and play with mazes.

Maze Visibility

After creating a maze, it doesn't automatically become public. You can make a request to make it public in the maze detail page, and it can be reviewed and made public by a moderator.

If you edit a maze, it becomes private if it was public before, and thus you need to request to make it public again.

Moderators can remove mazes from public view.
